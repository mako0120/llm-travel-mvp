あなたはプロのプレゼンテーションデザイナー兼リサーチャーです。以下の情報をもとに、
日本語のビジネス向けPowerPointプレゼンテーションを作成してください。

# テーマ
「ChatGPT Atlas、公開から1年足らずで終了。8月9日までにやるべきこと」
— AI内蔵ブラウザの短い生涯から学べることと、利用者が今すぐ確認すべき手順

# 想定視聴者
18〜45歳。ChatGPT Atlasを使ったことがある、または使っているかもしれない会社員・
個人事業主・学生。AI初心者にも分かるように、専門用語は初出時に短く説明する。

# 必ず使う事実(すべて出典付き。この範囲を超えて数字を創作しないでください)

- OpenAIが、ChatGPTを組み込んだAI内蔵Webブラウザ「ChatGPT Atlas」の提供を
  2026年8月9日に終了すると発表(出典: OpenAI公式ヘルプセンター
  https://help.openai.com/ja-jp/articles/20001371-%E3%83%96%E3%83%A9%E3%82%A6%E3%82%B6%E3%83%99%E3%83%BC%E3%82%B9%E3%81%AE%E3%82%A8%E3%83%BC%E3%82%B8%E3%82%A7%E3%83%B3%E3%83%88%E5%9E%8B%E4%BD%9C%E6%A5%AD%E3%81%AB%E5%90%91%E3%81%91%E3%81%9F-atlas-%E3%81%8B%E3%82%89-chatgpt-%E3%81%B8%E3%81%AE%E9%80%B2%E5%8C%96 )
- Atlasは2025年10月に公開されており、終了時点で公開から1年足らず(出典: ITmedia NEWS
  https://www.itmedia.co.jp/news/articles/2607/13/news090.html )
- 廃止理由は、ブラウザ由来のエージェント機能をChatGPT本体とCodexに移管するためと
  説明されている(出典: OpenAI公式ヘルプセンター)
- Atlasのブラウジング機能自体は消滅せず、新しいChatGPTデスクトップアプリの内蔵ブラウザと
  ChatGPT公式のChrome拡張機能に機能が移される(出典: 同上)
- ブックマークは自動移行されない。Cookie・ログインセッションも他ブラウザにインポート
  できない。8月9日より前に手動でのエクスポート・ログイン情報確認が必要(出典: ケータイWatch
  https://k-tai.watch.impress.co.jp/docs/news/2124737.html )

# 不明な点(正直に「不明」として扱ってください。推測で埋めないでください)

- Atlasの利用者数・利用実態
- 終了決定に至った具体的な内部要因

# スライド構成(目安15〜20枚。内容に応じて増減して構いません)

1. 表紙
2. 今回の結論(1年足らずで終了・8月9日締切・データは自動移行されない)
3. この動画/資料で得られること
4. なぜ今このテーマを扱うのか(締切が近い)
5. ChatGPT Atlasとは何だったか(2025年10月公開、基礎知識)
6. 何が終了するのか(公式発表の内容)
7. なぜ終了するのか(ChatGPT本体・Codexへの機能移管という理由)
8. 機能はどこへ行くのか(デスクトップアプリの内蔵ブラウザ・Chrome拡張機能)
9. 利用者が今すぐ確認すべきこと(ブックマーク・Cookie・ログインセッション)
10. 具体的な移行手順(締切までにやること、をステップで)
11. この出来事から見える示唆(専用ブラウザという形態の見直し、をメディアの論評として明記)
12. 注意点(不明点として明記した項目を含め、正直に伝える)
13. 反対意見・弱点(終了の内部要因は公式には明かされていない点)
14. まとめ・次のアクション(8月9日までに確認する、という具体的な行動喚起)
15. 出典一覧

# デザイン・品質ルール(必ず守ってください)

- 16:9のスライドサイズ
- 1スライド1メッセージ、本文は最大5ブレット、1ブレット2行以内
- 数字・日付を扱うスライドには出典を必ず明記する
- 全スライドにスピーカーノート(実際に話す内容の下書き、1枚100〜300字程度)を付ける
- 誇張表現は使わない
- 実在企業のロゴ・商標画像は使用しない

# 出力形式

Code Interpreter(Pythonの実行環境)が使える場合は、python-pptxライブラリを使って
実際に .pptx ファイルを生成し、ダウンロードリンクを提示してください。
使えない場合は、スライドごとに「見出し/本文/図解の指示/出典/スピーカーノート」を
テキストで出力してください。

最後に、上記「必ず使う事実」以外の数字を追加していないか、自己チェックした結果を
簡潔に報告してください。
