# 岡本和真選手、6月度ア・リーグ月間最優秀新人に選出(2026-07-24・定期実行27本目)

生成方式A(30枚・エンゲージメント設計ルール・初心者配慮ルール・文字を減らし図解で
伝えるルール)+ AI対話ナレーション・スライド連動モードで制作。

## 選定理由

- 対象ジャンル「海外で活躍する日本人」枠(20%)。定期実行24〜26本目が3回連続で
  AI/エージェント系のテーマだったため、ジャンル配分のバランスを取る目的も兼ねる
- 2026年7月3日、米大リーグ機構がブルージェイズの岡本和真選手を6月度ア・リーグ
  月間最優秀新人に選出。5月度の村上宗隆選手に続き2ヶ月連続で日本人打者が受賞
- NHK・日本経済新聞という信頼性の高い報道機関に加え、時事通信・Yahoo!ニュース
  (スポニチアネックス、サンケイスポーツ、ベースボールキング)等、複数の
  独立系メディアが報道しており事実確認が取れる

## 見送った候補

- ElevenLabsの「ElevenMusic」大型アップデート(2026-07-23) — 話題性・新しさは高いが、ジャンル配分のバランス上、次回以降に持ち越し
- Netchex「Mesh」(AI HR担当者チーム、2026-07-20) — 日本での知名度が低い米国のHR SaaS企業のため見送り
- 急成長SNSアカウント枠 — 具体的なアカウント名を確認できる情報源が見つからず、該当なしとして正直に報告

詳細な評価点は`ai-company-os/research/2026-07-24_theme-evaluation-round6.md`を参照。

## 適用したルール

- 30枚デッキ、bullets比率26.7%(40%上限クリア)、同一レイアウト連続なし
- 文字中心の説明はtable/diagram/bar_chart/comparisonで図解・グラフ化
  (6月の成績・受賞者比較は`table`/`comparison`、打率とOPSの比較は`bar_chart`、
  2ヶ月連続受賞の流れは`diagram`で表現)
- 初心者向け前提知識スライド + 「これは何を意味するか・どう楽しむか」の実践セクション
- 進捗バー・カード影・タイトルアクセントの視覚強化(`docs/04_PowerPoint.md`参照)
- 前サイクルとの接続スライドは設けていない
- 実在選手の写真・肖像は使用しない

## 検品結果

- `verify_pptx.py --min-slides 30 --max-slides 30` → 合格
- `build_deck.py`の同一レイアウト4連続以上チェック・bullets比率40%以下チェック(26.7%) → 合格
- `generate_dialogue_script.py --deck`のスライド数整合性検証(30枚、1〜30連番) → 合格
- 著作権・品質自己評価: 95点/100点(85点以上のため修正不要)

## リスク配慮

- 政治的テーマではなく、選手個人の表彰という事実紹介
- 対立を煽る表現・他選手比較の誇張を避け、推測部分は「推測」と明示している
- 実在選手名は事実として扱うが、写真・肖像・球団ロゴは使用しない

漢字の組み合わせ)があり、修正操作中にCanva接続が切断されたため、
`build_thumbnail.py`のPillow版(ルートB、hypeスタイル)に切り替えて完成させた。
再生時間バッジは、この時点では音声が未生成のため省略している(CI完了後、
実測値が判明次第追加する)。
