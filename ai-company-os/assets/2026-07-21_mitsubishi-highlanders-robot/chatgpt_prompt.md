# 必ず使う事実(すべて出典付き。この範囲を超えて数字を創作しないでください)

- 三菱自動車が2026年7月9日、東京大学発のロボット開発スタートアップHighlanders
  (2023年設立)と「国産人型ロボット」開発で協業すると発表(出典: 日本経済新聞
  https://www.nikkei.com/article/DGXZQOUC066YP0W6A700C2000000/ )
- 量産化に向け、2027年には月1000台製造できる体制を整える方針(出典: 日本経済新聞 /
  ITmedia AI+ https://www.itmedia.co.jp/aiplus/article/2607/09/2000000177/ )
- 三菱自動車の京都製作所での量産を目指し、2027年の早い段階で生産開始、同年後半を
  めどに自社工場の一部業務に導入予定(出典: 日本経済新聞)
- Highlandersは「労働をロボットで一掃する」というビジョンを掲げ、防衛・インフラ・
  製造分野で実証実験を重ねてきた(出典: ITmedia エグゼクティブ)
- 開発中の人型ロボット「N」は身長175cm・体重75kg、独自の5本指ハンドを持ち、
  深層学習で全身の関節を制御、バッテリーで約2時間連続稼働可能
  (出典: ITmedia AI+)
- 三菱自動車は外販も視野に入れていると報じられている(出典: 日本経済新聞)

# デザイン・品質ルール(必ず守ってください)

- 16:9のスライドサイズ
- 1スライド1メッセージ、本文は最大5ブレット、1ブレット2行以内
- 数字を扱うスライドには出典を必ず明記する
- グラフ・タイムラインはネイティブな図形機能で作成し、画像の貼り付けにしない

- 実在企業のロゴ・商標画像、ロボットの写真は使用しない(テキスト・自作図解のみ)
