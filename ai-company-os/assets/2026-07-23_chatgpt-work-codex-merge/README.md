# ChatGPTとCodexが、ついに1つに — OpenAI「Work」モード新設(2026-07-23・定期実行22本目)

生成方式A(30枚標準・エンゲージメント設計ルール)+ AI対話ナレーション・スライド連動モードで
制作。

## 選定理由

- 対象ジャンル「AI/エージェント系」枠(70%)
- OpenAIのChatGPTアプリとCodexアプリが1つの統合デスクトップアプリに再編され、新たに
  「Work」モードが追加されるという、プロダクト構成そのものの変化
- Neowin・TechTimes・quasa.io・Coursiv等、複数の独立系メディアが2026年7月9〜10日に
  一斉に報道しており、単一のSEOブログ発の憶測記事ではないことを確認した
- 前回テーマ(定期実行21本目「Buzz」)の「AIエージェントをメンバーとして扱う」という
  流れと接続する共通の切り口があり、シリーズとしての一貫性がある

## 見送った候補

- Grok 4.5のベンチマーク論争 — 独立系の言及元の多くがAI専門ブログ(コンテンツファーム
  寄り)で、幅広い一般メディアでの一次報道が確認できなかったため見送り
- 中国の新たなAI規制 — 詳細な一次情報にたどり着けず、政治・規制がらみで高リスクのため見送り
- Oracle AI Agent Studio・Black Lake TechnologiesのWAIC出展 — 企業向けニュースリリース
  の域を出ず、独立系メディアによる幅広い報道が確認できなかったため見送り
