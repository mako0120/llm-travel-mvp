# テーマ
「AI音声クローン詐欺、日本の遭遇率は世界最低の3%。それでも油断できない理由」
— マカフィーの7カ国調査から、日本の現状と今後の備えを読み解く

# 想定視聴者
18〜45歳。家族の安全を気にする会社員・個人事業主。特に、送金業務を一人で担う
個人事業主や、高齢の親を持つ層を意識する。セキュリティの専門知識は前提にしない。

# 必ず使う事実(すべて出典付き。この範囲を超えて数字を創作しないでください)

- マカフィーが7ヵ国の18歳以上の成人7,054人を対象にAI音声詐欺の調査を実施
  (出典: マカフィー公式プレスリリース
  https://prtimes.jp/main/html/rd/p/000000032.000033447.html )
- 「自身が遭遇した」は全体10%に対し、日本は3%で7カ国中最低(出典: マカフィー公式 /
  共同通信配信 https://kumanichi.com/articles/1366345 )
- 「知人が遭遇した」は全体15%に対し、日本は5%(出典: 同上)
- 7カ国合計727人が「自身が遭遇した」と回答、日本はそのうち最少の30人(出典: 共同通信配信)
- 遭遇者のうち77%が実際に金銭被害に遭ったと回答。一部は100万円以上の被害(出典: 共同通信配信)
- 日本は音声データのオンライン共有頻度も7カ国中最低(週1回以上共有28%、週5回以上5%、
  いずれも世界平均の半分程度)(出典: マカフィー公式)
- わずか3秒の音声(留守番電話・SNS動画・ポッドキャスト等)から約85%の精度で
  音声クローンが作成可能(出典: マカフィー学習コンテンツ)
- AI生成フィッシング詐欺全体の割合が、2025年11月〜12月の1か月で4%→56%に急増
  (約14倍)(出典: 財経新聞 https://www.zaikei.co.jp/article/20260708/860236.html )

# 不明な点(正直に「不明」として扱ってください。推測で埋めないでください)

- マカフィー調査の正確な実施月
- 日本国内での具体的な検挙事例・警察庁統計との突き合わせ
- 日本の遭遇率が低い理由の因果関係(音声共有頻度の低さとの相関は事実だが、
  因果は証明されていない)

# スライド構成(目安20〜25枚。内容に応じて増減して構いません)

1. 表紙
2. 今回の結論(日本は世界最低の3%、しかし77%が金銭被害という数字)
3. この動画/資料で得られること
4. なぜ今このテーマを扱うのか
5. マカフィー調査の概要(7カ国7,054人)
6. 日本の遭遇率を数字で見る(3% vs 世界平均10%、をグラフで)
7. 「遭遇したら77%が金銭被害」という重さ
8. なぜ音声クローンはこんなに簡単に作れるのか(3秒・85%精度の仕組みを基礎から説明)
9. なぜ日本の遭遇率は低いのか(音声データ共有頻度の低さとの相関、ただし因果は不明と明記)
10. 楽観できない理由(AIフィッシング全体が1か月で14倍という急増トレンド)
11. 個人事業主・家族が今すぐできる対策(電話以外の方法で本人確認する「合言葉」設定等、
    一般的に有効とされる対策を具体的に)
12. 注意点(不明点として明記した項目を含め、正直に伝える)
13. 反対意見・弱点(1社の調査であり、他機関の追試ではない点)
14. まとめ・次のアクション
15. 出典一覧

# デザイン・品質ルール(必ず守ってください)

- 16:9のスライドサイズ
- 1スライド1メッセージ、本文は最大5ブレット、1ブレット2行以内
- 数字を扱うスライドには出典を必ず明記する
- グラフはネイティブなグラフ機能で作成し、画像の貼り付けにしない
