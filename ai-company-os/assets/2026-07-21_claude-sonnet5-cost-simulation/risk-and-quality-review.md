# リスク・著作権チェックと品質自己評価

対象: Claude Sonnet 5 "エージェント運用費"は結局いくら?(30スライド版)

## 著作権・安全性チェック(仕様書13章準拠)

| 項目 | 結果 |
|---|---|
| 画像の利用条件 | 本デッキは図形・グラフのみで構成し、外部画像は使用していない(著作権リスクなし) |
| ロゴの利用条件 | Anthropic/Claudeのロゴは使用していない。社名・製品名はテキストのみで言及 |
| スクリーンショットの利用条件 | 使用していない |
| スポーツ映像の権利 | 該当なし(本テーマはスポーツを扱っていない。別候補の佐藤龍之介選手の件を採用していれば試合映像は不使用の方針だった) |
| 人物の顔・声の生成 | 行っていない。実在人物の合成音声・顔画像は一切生成していない |
| 記事の長文引用 | 各スライドは要約・独自の試算であり、報道記事の長文引用はしていない |
| AI生成画像の表示義務 | 本デッキに画像生成AIによる素材は含まれていない(python-pptxによる図形描画のみ) |
| プラットフォームの合成コンテンツ表示 | 該当する合成コンテンツなし |

**結論: 著作権・安全性の観点で公開を妨げる問題は確認されなかった。**

## 人間承認ポイント(仕様書16章)の該当確認

以下のいずれにも該当しないため、通常フロー(自動制作)の範囲内と判断した。ただし**公開・投稿自体は仕様書どおり人間承認後に行う**。

- [ ] 本人や公式による確認が取れない → 該当なし(Anthropic公式・ソフトバンク公式・TechCrunchの一次/準一次情報を使用)
- [ ] 著作権侵害の可能性 → 該当なし(上表のとおり)
- [ ] 政治・医療・投資・法律等の高リスク領域 → 該当なし(ITツールの料金解説)
- [ ] 有名人の噂・私生活 → 該当なし
- [ ] 情報源同士が矛盾する → 該当なし(価格情報はAnthropic公式とClaude Platform Docsで一致)
- [ ] AIによる実在人物の顔・声の生成 → 該当なし(生成していない)
- [ ] 公開により誰かに重大な不利益を与える可能性 → 該当なし

## 検査項目(仕様書14章)

- [x] 出典のない数字がないか → すべての数字にスライド内または本ファイル群に出典を明記(試算値は「公表レートに基づく仮定計算」と明記)
- [x] 公開日と出来事の日を混同していないか → 各出典に公開日・発表日を区別して記載
- [x] 古いモデルや料金を最新と記載していないか → 2026年7月21日時点の情報で統一、9月以降の価格変更も明記
- [x] 同じ主張を繰り返していないか → 「8/31締切」「トークナイザー変更」は複数スライドで言及するが、都度異なる角度(基礎知識→試算→注意点→まとめ)で展開
- [x] 文字だけのスライドが多くないか → 30枚中、表4枚・カード5枚・比較2枚・グラフ1枚・アジェンダ1枚で視覚要素を分散
- [x] 競合動画の単なる言い換えになっていないか → 差別化方針(研究レポート参照)のとおり、料金表の紹介に留まらない独自試算
- [x] AI量産コンテンツに見えないか → 実データに基づく独自試算・出典明記により差別化
- [x] 視聴者が実行できる具体策があるか → CTAで「8月中に自分の利用量で試算する」という具体的行動を提示
