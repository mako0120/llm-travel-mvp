## メタ情報

- デッキ名: OpenAI、企業向けAIエージェント基盤「Presence」を発表(定期実行23本目)
- 元になる PowerPoint / 構成 Markdown: `deck_spec.json` / `deck.pptx`(30枚)
- 目的・想定視聴者: AI/エージェント系のニュースに関心があるビジネスパーソン・意思決定者・
  一般視聴者向け。専門用語(AIエージェント・エージェント基盤・ガードレール等)を
  知らない初心者にも分かるよう構成している
- パレット(HEX、navy_goldプリセット): Primary `#0F1E3D` / Base `#F5F1E6` / Accent `#C9A24B`
- フォント指定(Canva 代替): 見出し・本文ともに Noto Sans JP(pptx版は既定フォントを使用)
- アニメーション: なし
- 使用素材のライセンス確認: 無料素材のみ(実在企業のロゴ・商標画像は使用しない、
  図形・アイコンはCanva内蔵の無料素材のみを使う)

## レイアウト種別 → Canva要素の対応(スライド全体で共通)

| pptx側レイアウト | Canva要素 |
|---|---|
| title | タイトルスライド(大見出し+サブコピー+出典フッター) |
| agenda | 見出し+丸数字バッジ付きリスト(ダーク背景) |
| bullets | 見出し+ブレットリスト(最大5項目) |
| cards | カードグリッド(2〜4枚、ラベル+短文) |
| big_stat | 1数字強調レイアウト(ダーク背景、中央に巨大数字) |
| diagram | ノード(角丸ボックス)+矢印コネクタ(2〜3ノード) |
| steps | 横並びステップカード+チェブロン矢印 |
| table | 表(ヘッダー行+データ行) |
| comparison | 2カラム比較(左右カード+リスト) |
| cta | CTAスライド(大見出し+サブ+アクション文、ダーク背景) |

## 移植手順(人間向け)

1. Canva で 16:9 プレゼンテーションを新規作成
2. ブランドカラーに上記 HEX を登録
3. 上表のレイアウト種別ごとに「レイアウト種別 → Canva要素の対応」表の要素を使い、
   `deck_spec.json`の該当スライドから見出し・本文・出典テキストをそのままコピーする
4. 完成後、PowerPoint 版と並べて構成・数字の一致を確認
