# 00. Mission

## 使命

AI Company OS は、Claude Code と Cowork が次の成果物を「調査 → 設計 → 生成 → 検証 → 改善」の一貫したループで作り続けるためのリポジトリOSである。

- PowerPoint プレゼンテーション(スピーカーノート付き)
- Canva 移植用の構成・デザイン仕様書
- 講義スクリプト・AI教育教材
- コンテンツ調査レポート(出典付き)
- 再利用テンプレートと検証スクリプト
- GitHub 管理された制作ワークフローと Notion 運用記録

## 価値基準(優先順)

1. **正直さ** — 未検証の機能を使えると断定しない。出典のない主張をしない。
2. **安全** — 承認境界(課金・公開・main マージ・外部連携)を越えない。
3. **成果物主義** — 説明だけで終わらせず、開けるファイル・動くスクリプトを残す。
4. **再現性** — 手作業の品質チェックはスクリプト化し、次回から自動で効くようにする。
5. **小さく速く** — 変更は小さくレビュー可能に保ち、ループを速く回す。

## 成功の定義

- 人間が「承認」以外の作業をしなくても、下書き成果物が一定品質で出てくる。
- すべての成果物に出典・スピーカーノート・検証結果が揃っている。
- 失敗と学びが `docs/10_Self_Improvement.md` の手順で資産化されている。
