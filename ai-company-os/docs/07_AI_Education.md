# 07. AI Education(AI教育教材の標準)

## 教材の種類

- 講義スライド(PowerPoint 標準に準拠、スピーカーノート=講義スクリプト)
- ハンズオン手順書(コピペで動くコマンド、期待される出力、失敗時の対処)
- 理解度確認(各章末に3〜5問。答えと解説付き)

## 設計規則

- 対象レベルを最初に宣言する: 入門 / 初級 / 中級 / 上級。
- 1教材1スキル。「AIとは何か」のような広すぎるテーマは分割する。
- 専門用語は初出時に1行で定義する。
- 実演は必ず本物のツールで検証してから教材化する(動かないハンズオンを配布しない)。
- 料金・制限・規約に関する記述は公式ドキュメントを出典にし、更新日を記載する(変化が速いため)。

## 品質チェック

- [ ] 対象レベルと前提知識が明記されている
- [ ] すべてのコマンド・手順を実環境で検証した
- [ ] スクリーンショットや図はオリジナル(他者の教材からコピーしない)
- [ ] 理解度確認問題に解答・解説がある
- [ ] 出典と最終検証日が記載されている

## 配布

教材の公開・販売・SNS発信は承認境界(`docs/01_Core_Rules.md`)に従い、人間承認後に行う。
