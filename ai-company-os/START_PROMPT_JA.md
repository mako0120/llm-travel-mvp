# Claude Code 実行指示

このリポジトリの `CLAUDE.md` を最初に読み、続けて `docs/00_Mission.md` から `docs/12_Project_Rules.md` まで順番に読んでください。

その後、次を実行してください。

1. 現在利用できるスキル、MCP、コネクタ、コマンド、ライブラリ、権限を確認する。
2. PowerPoint・Canva・Claude Code・Coworkで利用できるものだけを分類する。
3. 人気のある手法だけでなく、信頼性のある「穴場」のスキル、MCP、ライブラリ、テンプレートも公式情報・一次情報・GitHub等から調査する。
4. 各候補について、提供元、用途、導入条件、ライセンス、更新状況、セキュリティ、期待効果、リスクを比較する。
5. 未確認の機能を利用可能だと断定しない。
6. 外部連携のインストール、OAuth、課金、mainへのマージ、本番変更は行わず、提案と検証計画までに留める。
7. 現在のリポジトリを監査し、不足するディレクトリ、テンプレート、検証スクリプト、GitHub Actions、ドキュメントを提案する。
8. 最初の実装候補を3件に絞り、効果・難易度・リスクで順位付けする。
9. 最有力案についてIssue草案、ブランチ名、実装計画、テスト計画、成果物、承認事項を出す。
10. 人間の承認を待つ。

出力は `docs/11_Output_Format.md` に従ってください。
